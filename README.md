## 進度：

✅A3 - 元件拆分<br>
✅A4 - 元件渲染<br>
✅A5 - 元件實作<br>
A6 - 表單實作<br>

## 開始使用

1. Clone 這支 Repository

```
git clone https://github.com/mirageapache/s3F_Alpha-shop.git
```

2.安裝 npm

```
npm install
```

3.啟動程式

```
npm start
```

4.瀏覽網頁

```
開啟瀏覽器並輸入http://localhost:3000，即可瀏覽網頁
```

5.終止程式

```
在終端機按下兩次 ctrl + C 即可終止程式
```

## 環境設置：

- node.js - 18.12.1
- npm - 8.19.2
- React - 18.2.0
