# ALPHA_Shop 專案

## 進度：

### ✅A3 - 元件拆分

### A4 - 元件渲染

### A5 - 元件實作

### A6 - 表單實作

## 使用方式：

### 用終端機下載專案 git clone https://github.com/mirageapache/s3F_A3-Alpha-shop.git

### 進入專案資料夾內，並安裝 npm (npm install)

### 再執行專案 - npm start

### (在瀏覽器網址輸入 'http://localhost:3000/') - 完成

## 環境設置：

### node.js - 18.12.1

### npm - 8.19.2

### React - 18.2.0
